# Store Rating App

A full-stack role-based Store Rating platform.

## Tech Stack
- React.js
- Tailwind CSS
- Node.js
- Express.js
- PostgreSQL
- Sequelize
- JWT Authentication

## Features

### Admin
- Manage users
- Manage stores
- Assign store owners

### User
- Browse stores
- Submit ratings
- Update/Delete ratings

### Store Owner
- View owned stores
- View ratings analytics
